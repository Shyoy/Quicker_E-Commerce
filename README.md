# Getting Started

<br>


## Initial Setup
### Backend:
1. **Create virtualenv**:
    ```cmd
    cd .\Back
    ```
    ```cmd
    python -m venv venv
    ```
2. Install all needed requirements
    ```cmd
    pip install -r requirements.txt
    ```
3. **Run the server**:
    ```cmd
    python app.py
    ```
### Frontend:
1. **Open new terminal**:
    ```cmd
    cd .\Front\REACT\ayp
    ```
2. Install all needed requirements
    ```cmd
    npm i
    ```
    
2. **Run the server**:
    ```cmd
    npm start
    ```
